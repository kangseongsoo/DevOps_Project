# 학습형 AI 챗봇 서비스 프로젝트 로드맵

## 1단계: 초기 기획 및 환경 설정

* 프로젝트 요구사항 명세 작성
* 개발환경 구축 (FastAPI, React, Docker, MariaDB, Gitea)
* 개발 및 운영환경 분리 설정

## 2단계: AI 서비스 백엔드 개발

* LangChain, LangGraph 이용한 챗봇 알고리즘 설계
* GPT API 연동 및 API 키 관리
* FastAPI를 활용한 챗봇 API 구축 및 기본 동작 테스트
* Docker 컨테이너화 (챗봇 서비스)

## 3단계: 사용자 관리 및 데이터 관리 시스템 구축

* MariaDB 스키마 설계 및 구축
* 사용자 인증 및 권한 관리 모듈 개발 (JWT 인증)
* 학습 데이터 메타데이터 관리 기능 구현
* Docker 컨테이너화 (MariaDB 서비스)

## 4단계: 사용자 UI 개발

* React를 이용한 사용자 인터페이스 설계
* 챗봇 API와의 통신 기능 구현 (Axios 또는 Fetch API)
* 사용자 피드백 및 인터랙티브 기능 개발
* Docker 컨테이너화 (React 앱)

## 5단계: 통합 테스트 및 최적화

* API 및 UI 통합 테스트
* 성능 및 부하 테스트
* 서비스 최적화 및 리팩토링
* CI/CD 파이프라인 구축 (Gitea Actions 및 Jenkins)

## 6단계: 모니터링 및 배포 자동화

* 프로메테우스 및 그라파나 설치 및 설정
* 시스템 및 서비스 상태 모니터링 대시보드 구축
* 장애 알림 설정 및 자동 복구 프로세스 설정
* 배포 자동화 환경 구축 및 운영

## 7단계: 출시 및 운영 관리

* 최종 배포 및 서비스 런칭
* 운영 환경에서의 실시간 모니터링
* 사용자 피드백 반영 및 지속적 업데이트
* 장애 대응 및 유지보수 관리
